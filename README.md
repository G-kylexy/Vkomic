# Vkomic
Une application qui se branche sur https://vk.com/board203785966 et permet de télécharger les BD, mangas et comics disponibles.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. (Optionnel) Créez un fichier `.env.local` si vous avez d'autres variables à injecter
3. Run the app: `npm run dev`
